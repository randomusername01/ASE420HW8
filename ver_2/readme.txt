# Time Management System

## Overview
This project implements a comprehensive time management system, designed to track, manage, and report on various tasks and activities. With functionalities to handle task records, search and query data, and generate detailed reports, this system is an essential tool for personal and professional time management.

## Features
- **Task Recording**: Log tasks with associated dates, times, and descriptions.
- **Data Query**: Search through recorded tasks based on specific criteria.
- **Reporting**: Generate reports showing completed tasks within a date range, and aggregate time spent on tasks grouped by tags.

## Getting Started
To use this system, follow these steps:
1. Clone the repository.
2. Install required dependencies.
3. Run `ver_2.py` to start the application.
4. Follow on-screen instructions to interact with the system.
